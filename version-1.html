0.1

This is the first iteration of the project. I started by knowing no HTML, CSS and only vary basic and theoretical JS (from the textbook Eloquent Javascript). My first main source was a tutorial by the name HTML And CSS is Hard by Interneting Is Hard. While it may sound dubious at first sight, it did a great job introducing me to basic web development. The first visual prototype was built in Adobe XD:

This was translated to HTML and CSS by using floats. Starting from version 0.2, the page uses flexbox. It offed only a static page with no buttons and no interactions. The page can be accessed here (opens in new tab).
